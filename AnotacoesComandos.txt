Comandos Angular CLI
C:\> ng new <nome-projeto>  - criação de projeto 
C:\> ng serve  - compila e serve aplicação no endereço patrão localhost://4200
C:\> ng g c <nome-do-component> - criando componente
C:\> ng g m <nome-do-modulo> - criando um novo modulo 
C:\> ng g s <nome-do-servico> - criando um servico
C:> node -v   - mostra a versão do node { requerido para instalação do pacote cli }
C:> npm install angular-cli  - instalar o angular-cli

usando pré-processador de css
